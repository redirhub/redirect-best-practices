# Contributing to Redirect Best Practices

Contributions welcome. Here's how:

1. Keep suggestions focused on redirect best practices (types, SEO, performance, security, testing)
2. Link to high-quality, authoritative sources (Google, MDN, OWASP, official docs, well-researched industry articles)
3. Avoid duplicating content already covered in other sections
4. Submit a pull request with a short explanation of what you're adding and why it matters

When adding links, prefer official documentation and primary sources over aggregated listicles.
